# tables

This was an experiment in porting the Subversion repository of the tables package from R-forge to Github.

I have since found out that someone else is already doing this, so I've dropped my attempt at mirroring.

To view tables sources, go to 

https://github.com/rforge/tables/tree/master/pkg

To install tables from Github, use

devtools::install_github("rforge/tables", subdir="pkg/tables")

For now, the master copy remains on R-forge.
